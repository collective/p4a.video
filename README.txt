p4a.video
=========

Overview
--------

The *p4a.video* package provides a framework for handling video content on
the Zope 2 and Zope 3 platforms.  It was inspired by the Plone ATvideo
product and even borrows some UI.

Developer Notes
---------------

For implementing a new media player for a particular mime type please
see *media-player.txt*.  To run the tests you will need to ensure
p4a.video is available in your ``$INSTANCE/lib/python`` directory.
